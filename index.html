About

TwitchWatch2 is a cross-platform desktop application geared towards twitch.tv notifications. Here's some features in a pretty list

  • Streamer status
  • Win/Osx/Lin desktop notifications
  • Open stream in browser
  • Open stream with livestreamer (requires livestreamer to be installed)
